## Stripe and Borders

`stripe` controls the background color of even rows; `bordered` supports both boolean and object forms â€” the object form lets you customize vertical borders via `color`, `width` and `style`:

::component-example
---
name: DataTableStripeBorderedExample
options:
  - name: stripe
    label: stripe
    items: [true, false]
    default: true
  - name: bordered
    label: bordered
    items:
      - true
      - false
      - { color: 'var(--ui-primary)', width: '2px', style: 'dashed' }
    default: true
---
::

## `density` Visual Density

`density` switches between three cell-padding presets: `compact`, `normal` and `comfortable`:

::component-example
---
name: DataTableDensityExample
options:
  - name: density
    label: density
    items:
      - compact
      - normal
      - comfortable
    default: normal
---
::

## `fitContent` Fit-Content Width

`fitContent` makes the table width determined by column widths rather than stretching to fill the parent container â€” suitable for compact utility tables:

::component-example
---
name: DataTableFitContentExample
props:
  class: w-full
options:
  - name: fit-content
    label: fitContent
    items:
      - true
      - false
    default: true
---
::

## `emptyCell` Empty-Value Placeholder

`emptyCell` supports a string, a function template (returning a VNode) and `false`, respectively for custom text, a custom node and disabling the placeholder entirely. A per-column `emptyCell` takes precedence over the global setting:

::component-example
---
name: DataTableEmptyCellVariantsExample
options:
  - name: empty-cell
    label: emptyCell
    items:
      - 'â€”'
      - 'N/A'
      - 'None'
      - false
    default: 'â€”'
---
::

## `sticky` Sticky Header

`sticky` is enabled by default; disabling it makes the header scroll with the content. It requires a scroll container with a constrained height (e.g. `ui.root: 'max-h-[40vh]'`):

::component-example
---
name: DataTableStickyExample
options:
  - name: sticky
    label: sticky
    items:
      - true
      - false
    default: true
---
::
