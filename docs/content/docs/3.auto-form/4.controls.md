每种 Zod 类型都有默认控件（见 [API 与配置 · 控件类型映射](/docs/auto-form/api#控件类型映射)）。通过 `meta({ type })` 或 `afz.xxx({ type })` 可切换到该类型支持的其他控件，并用 `controlProps` 透传控件属性。
