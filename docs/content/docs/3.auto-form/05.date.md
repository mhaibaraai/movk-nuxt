## ISO 格式字符串

对于需要 ISO 标准格式字符串的场景，提供三种工厂方法：

- `afz.isoDate()` - 验证 `YYYY-MM-DD` 格式
- `afz.isoTime()` - 验证 `HH:MM[:SS[.s+]]` 格式
- `afz.isoDatetime()` - 验证 ISO 8601 完整格式
