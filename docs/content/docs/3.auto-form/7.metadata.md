## 字段元数据

`meta()` 配置字段的 UI 属性：`label`、`description`、`hint`、`help`、`error`、`size` 等基础项，`class`/`ui` 自定义样式，`if`/`hidden` 控制条件渲染，`fieldSlots` 注入插槽，对象字段的 `collapsible` 渲染为可折叠分组。
