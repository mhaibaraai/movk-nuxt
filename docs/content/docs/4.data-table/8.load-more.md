## 触底加载

传入 `load-more` 回调即启用无限滚动模式（自动隐藏内置分页）。回调返回 Promise 时，组件在加载期间自动派生 `loading`；`can-load-more` 控制是否还能继续加载；`load-more-distance` 设置触发加载的距底像素阈值（默认 100）：

::component-example
---
name: DataTableLoadMoreExample
---
::

::note
需要一个受限高度的滚动容器（如 `ui.root: 'h-[60vh]'`）才能产生滚动并触发触底加载。若希望由调用方接管 loading，可让 `load-more` 同步返回（不返回 Promise）并自行传入 `:loading`。
::
