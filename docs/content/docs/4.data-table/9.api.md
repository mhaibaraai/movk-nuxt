## Expose

通过 [`useTemplateRef`](https://vuejs.org/api/composition-api-helpers.html#usetemplateref) 访问类型化的组件实例（`DataTableExposed<T>`）。

| Name | Type |
| ---- | ---- |
| `tableRef`{lang="ts-type"} | `HTMLTableElement \| null`{lang="ts-type"} <br> <div class="text-toned mt-1"><p>原生 table 元素</p></div> |
| `tableApi`{lang="ts-type"} | `Table<T> \| null`{lang="ts-type"} <br> <div class="text-toned mt-1"><p>TanStack Table 实例</p></div> |
| `el`{lang="ts-type"} | `HTMLElement \| null`{lang="ts-type"} <br> <div class="text-toned mt-1"><p>UTable 根元素（滚动容器）</p></div> |
| `scrollToTop(options?)`{lang="ts-type"} | `void`{lang="ts-type"} <br> <div class="text-toned mt-1"><p>滚动到顶部</p></div> |
| `clearSelection()`{lang="ts-type"} | `void`{lang="ts-type"} <br> <div class="text-toned mt-1"><p>清空行选择</p></div> |
| `expandToDepth(depth)`{lang="ts-type"} | `void`{lang="ts-type"} <br> <div class="text-toned mt-1"><p>展开到指定层级，`depth=0` 等价收起全部</p></div> |
| `collapseAll()`{lang="ts-type"} | `void`{lang="ts-type"} <br> <div class="text-toned mt-1"><p>收起全部行</p></div> |
| `treeSelection`{lang="ts-type"} | `TreeSelectionResult<T>`{lang="ts-type"} <br> <div class="text-toned mt-1"><p>树形选中派生：`selected`、`leaves`、`parents`、`halfSelected`、`strictlyChecked`</p></div> |

## 列类型

`columns` 数组的每一项是 `DataTableColumn<T>`，按形态区分：

| 类型 | 判定 | 说明 |
| --- | --- | --- |
| `DataTableDataColumn<T>`{lang="ts-type"} | 含 `accessorKey` | 数据列：排序、固定、截断、tooltip、可见性、自定义渲染 |
| `DataTableGroupColumn<T>`{lang="ts-type"} | 含 `children` | 分组表头 |
| `DataTableSelectionColumn<T>`{lang="ts-type"} | `type: 'selection'` | 选择列（单/多选、树形策略） |
| `DataTableIndexColumn<T>`{lang="ts-type"} | `type: 'index'` | 索引列 |
| `DataTableExpandColumn<T>`{lang="ts-type"} | `type: 'expand'` | 展开列 |
| `DataTableRowPinningColumn<T>`{lang="ts-type"} | `type: 'row-pinning'` | 行固定列 |
| `DataTableActionsColumn<T>`{lang="ts-type"} | `type: 'actions'` | 操作列 |

## 事件处理类型

事件处理器使用以下类型，参数由 contextual typing 推断：

| 类型 | 用途 |
| --- | --- |
| `DataTableSelectHandler<T>`{lang="ts-type"} | `@select` |
| `DataTableHoverHandler<T>`{lang="ts-type"} | `@hover` |
| `DataTableContextmenuHandler<T>`{lang="ts-type"} | `@row-contextmenu` |
