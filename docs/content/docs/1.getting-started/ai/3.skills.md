## 什么是 Agent Skills？

Agent Skills 是按 [Agent Skills 规范](https://agentskills.io/specification) 编写的 `SKILL.md` 结构化知识文件。与 [MCP Server](/docs/getting-started/ai/mcp)（提供实时文档检索工具）不同，Skills 直接加载进 AI 编码代理的上下文，让代理在整段对话中持续参考。

Movk Nuxt 提供单一 `movk-nuxt` 技能，覆盖 AutoForm、API、DataTable、独立组件与 Composables 五大能力域，注入「在何种场景使用哪个能力、如何正确书写」的指引，让生成的代码符合本库约定。

## 使用

`movk-nuxt` 技能让 AI 代理掌握构建 Movk Nuxt 应用的完整知识：

- **AutoForm**：Zod Schema 驱动表单生成、字段控件注册、布局与条件渲染
- **API**：`useApiFetch` 多端点、自动认证、业务码处理、上传/下载进度
- **DataTable**：列配置、树形数据、选择/展开、分页与加载更多
- **独立组件**：DatePicker、SearchForm、StarRating、PillGroup 等 M 前缀组件
- **Composables**：`useDateFormatter`、`useTheme`、`useMessageBox` 等工具函数

技能入口 `SKILL.md` 承载跨域铁律与路由表，按域拆分的细则位于 `references/`，由代理按需加载，确保响应聚焦且上下文高效。

::note
安装后，可在 Agent 聊天中输入 `/movk-nuxt` 触发技能。
::

### Skills CLI

[`skills`](https://skills.sh) CLI 是安装 Movk Nuxt 技能的最便捷方式，支持 35+ 代理，包括 Cursor、Claude Code、Codex、Windsurf、Cline 等。直接传入文档站地址即可，CLI 会自动抓取技能：

```bash
npx skills add https://nuxt.mhaibaraai.cn
```

通过 `--agent` 指定目标代理：

```bash
npx skills add https://nuxt.mhaibaraai.cn --agent cursor
npx skills add https://nuxt.mhaibaraai.cn --agent claude-code
```

或使用 `--global` 全局安装，让技能在所有项目中可用：

```bash
npx skills add https://nuxt.mhaibaraai.cn --global
```

::tip
访问 [skills.sh](https://skills.sh) 了解 Skills 生态并浏览更多可用技能。
::

### Cursor

#### 一键安装

点击下方按钮，直接在 Cursor 中安装 Movk Nuxt 技能：

::u-button
---
to: "cursor://anysphere.cursor-deeplink/install-skill?url=https://github.com/mhaibaraai/movk-nuxt/tree/main/docs/skills/movk-nuxt"
label: 安装 Skill
color: neutral
icon: i-simple-icons-cursor
---
::

#### 手动配置

1. 打开 Cursor，进入「Settings」>「Skills」
2. 点击「Add skill」，粘贴以下 URL：

```text
https://github.com/mhaibaraai/movk-nuxt/tree/main/docs/skills/movk-nuxt
```

### Claude Code

::note
确保已安装 Claude Code。参阅 [Anthropic 文档](https://docs.anthropic.com/en/docs/claude-code/quickstart) 获取安装说明。
::

通过 CLI 命令添加技能：

```bash
claude skill add https://github.com/mhaibaraai/movk-nuxt/tree/main/docs/skills/movk-nuxt
```
