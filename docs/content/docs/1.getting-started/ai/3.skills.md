## 什么是 Agent Skills？

Agent Skills 是按 [Agent Skills 规范](https://agentskills.io/specification) 编写的 `SKILL.md` 技能包。与 [MCP Server](/docs/getting-started/ai/mcp)（提供实时文档检索）互补，Skills 直接向 AI 编码代理注入「在何种场景使用 Movk Nuxt 的哪个能力、如何正确书写」的指引，让生成的代码符合本库约定。

Movk Nuxt 按功能域提供以下技能：

| Skill | 适用场景 |
| --- | --- |
| `auto-form` | 用 Zod Schema 驱动生成表单、字段控件、布局与校验 |
| `api` | 数据请求、多端点、认证、业务码、Toast、上传/下载进度 |
| `data-table` | 列配置、树形数据、选择/展开、分页与加载更多 |
| `components` | 独立 UI 组件（DatePicker、SearchForm、StarRating 等） |
| `composables` | useDateFormatter、useTheme、useMessageBox 等工具函数 |

## 安装

技能位于仓库的 [`docs/skills`](https://github.com/mhaibaraai/movk-nuxt/tree/main/docs/skills) 目录。将所需技能目录复制到项目或个人技能目录即可：

```bash
# 项目级（随仓库共享）
mkdir -p .claude/skills
cp -r docs/skills/auto-form .claude/skills/

# 个人级（仅本机）
cp -r docs/skills/auto-form ~/.claude/skills/
```

::tip{icon="i-lucide-plug" to="/docs/getting-started/ai/mcp"}
建议同时配置 MCP Server，让代理在使用技能时按需检索最新文档与示例。
::
