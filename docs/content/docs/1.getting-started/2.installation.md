## 环境要求

在开始之前，请确保您的开发环境满足以下要求：

- **Node.js**: >= 18.0.0
- **Nuxt**: >= 4.2.0
- **pnpm** (推荐) 或 npm/yarn

## 安装步骤

### 1. 创建 Nuxt 项目

如果您还没有 Nuxt 项目，可以使用以下命令创建一个新项目：

```bash [Terminal]
npx nuxi init my-nuxt-app
cd my-nuxt-app
```

### 2. 安装依赖

```bash [Terminal]
# 安装 Movk Nuxt 模块
pnpm add @movk/nuxt

# 安装必需的依赖
pnpm add @nuxt/ui zod
```

### 3. 配置模块

在 `nuxt.config.ts` 中添加模块：

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@movk/nuxt'],

  // 可选：配置模块选项
  movk: {
    prefix: 'M' // 组件前缀，默认为 'M'
  }
})
```

### 4. 导入 CSS

在您的 CSS 文件中导入必要的样式：

```css [app/assets/css/main.css]
@import '@nuxt/ui';
```

## 快速开始

安装完成后，创建您的第一个表单：

```vue [pages/index.vue]
<script setup lang="ts">
import { ref } from 'vue'
import { z } from 'zod/v4'
import { afz } from '@movk/nuxt/autoform'

const { afz } = useAutoForm()

const schema = afz.object({
  username: afz.string().min(3),
  email: afz.email(),
  age: afz.number().min(18)
})

const form = ref({})

async function handleSubmit(data) {
  console.log('表单数据:', data)
}
</script>

<template>
  <div class="container mx-auto p-4">
    <MAutoForm :schema="schema" :state="form" @submit="handleSubmit" />
  </div>
</template>
```

## 开发模式

启动开发服务器查看效果：

```bash [Terminal]
pnpm dev
```

现在访问 `http://localhost:3000` 即可看到您的第一个自动表单！

## 作为 Layer 使用

如果您想在自己的 Nuxt 项目中使用 Movk Nuxt 作为 layer：

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  extends: ['@movk/nuxt'],
  // 其他配置...
})
```

## 常见问题

### Q: 组件前缀可以修改吗？

A: 可以。在 `nuxt.config.ts` 中通过 `movk.prefix` 选项修改：

```ts
export default defineNuxtConfig({
  movk: {
    prefix: 'MyForm' // 使用 'MyForm' 前缀
  }
})
```

### Q: 如何自定义控件？

A: 详见 [自定义控件](/docs/advanced/custom-controls) 文档。

### Q: 支持哪些 Zod 类型？

A: 支持所有 Zod v4 内置类型，详见 [字段类型](/docs/field-types) 文档。
