## 什么是 Movk Nuxt？

Movk Nuxt 是一个基于 [Nuxt 4](https://nuxt.com) 的表单生成模块，它能够将 [Zod](https://zod.dev) Schema 自动转换为美观、交互式的表单界面。该模块构建在 [Nuxt UI](https://ui.nuxt.com) 组件库之上，提供开箱即用的表单解决方案。

## 核心优势

- **类型安全**: 基于 Zod Schema 的完整类型推断，编译时即能发现类型错误
- **高效开发**: 从 Schema 到表单的无缝转换，大幅提升开发效率
- **灵活布局**: 支持 `$layout` 标记创建复杂表单布局
- **丰富控件**: 内置 20+ 种表单控件，覆盖大部分业务场景
- **响应式设计**: 自动适配各种屏幕尺寸，提供最佳用户体验
- **完全可定制**: 支持自定义控件映射、元数据配置、表单 slots 等

## 快速预览

```typescript
import { z } from 'zod/v4'
import { afz } from '@movk/nuxt/autoform'

// 定义 Schema
const schema = afz.object({
  username: afz.string().min(3),
  email: afz.email(),
  age: afz.number().min(18)
})
```

```vue
<template>
  <MAutoForm :schema="schema" :state="form" @submit="handleSubmit" />
</template>
```

这就是创建一个完整表单所需的全部代码！

## 适用场景

- 用户注册和登录表单
- 产品信息录入表单
- 问卷调查和数据收集
- 管理后台的数据编辑页面
- 任何需要表单交互的场景

## 技术栈

- [Nuxt 4](https://nuxt.com) - 现代化的 Vue.js 框架
- [Nuxt UI](https://ui.nuxt.com) - 优雅的 UI 组件库
- [Zod v4](https://zod.dev) - 类型安全的 Schema 验证库
- [TypeScript](https://www.typescriptlang.org) - 提供完整的类型支持
