## 介绍

Movk Nuxt 是一个专为 Nuxt 4 设计的功能增强模块，提供一系列开箱即用的组件和工具函数，帮助你快速构建现代 Web 应用。

构建在 [Nuxt UI](https://ui.nuxt.com) 之上，Movk Nuxt 遵循相同的设计原则：美观、可访问、开发者友好。

::callout{icon="i-lucide-rocket" color="primary"}
当前版本聚焦于 **AutoForm** - 基于 Zod Schema 的自动表单生成系统。后续将扩展更多实用功能，包括 **AutoTable**、**UseApi** 等常用 composables。
::

## 核心特性

::card-group
  :::card{icon="i-lucide-zap" title="零模板代码"}
  通过 Zod Schema 定义数据结构，自动生成完整表单界面，无需编写重复的表单模板代码。
  :::

  :::card{icon="i-lucide-shield-check" title="类型安全"}
  完整的 TypeScript 类型推断，从 Schema 到表单数据全程类型安全。
  :::

  :::card{icon="i-lucide-paintbrush" title="灵活定制"}
  支持 15+ 种控件类型，可自由配置样式和行为，满足各种表单需求。
  :::

  :::card{icon="i-lucide-blocks" title="模块化架构"}
  采用模块化设计，未来将持续扩展更多实用组件和工具函数。
  :::
::

## 技术栈

Movk Nuxt 基于以下现代技术构建：

::note{icon="i-lucide-package"}
**核心依赖**

- [**Nuxt UI**](https://ui.nuxt.com) - 提供美观、可访问的基础组件
- [**Zod v4**](https://zod.dev) - Schema 验证和类型推断
- [**VueUse**](https://vueuse.org) - 实用的 Vue 组合式函数集合
- [**@movk/core**](https://github.com/movk/core) - Movk 核心工具库
::
