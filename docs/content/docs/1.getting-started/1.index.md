## 介绍

Movk Nuxt 是一个专为 Nuxt 4 设计的功能增强模块，提供一系列开箱即用的组件和工具函数，帮助你快速构建现代 Web 应用。

构建在 [Nuxt UI](https://ui.nuxt.com) 之上，Movk Nuxt 遵循相同的设计原则：美观、可访问、开发者友好。

::callout{icon="i-lucide-rocket" color="primary"}
当前版本聚焦于 **AutoForm** - 基于 Zod Schema 的自动表单生成系统。后续将扩展更多实用功能，包括 **AutoTable**、**UseApi** 等常用 composables。
::

## 当前功能

### AutoForm

基于 Zod Schema 的自动表单生成系统，通过声明式的 Schema 定义自动生成完整表单界面。

::code-group
```vue [基础示例]
<script setup lang="ts">
const { afz } = useAutoForm()

const schema = afz.object({
  username: afz.string().meta({
    label: '用户名',
    placeholder: '请输入用户名'
  }),
  email: afz.email().meta({
    label: '邮箱',
    placeholder: 'example@email.com'
  }),
  age: afz.number().min(18).meta({
    label: '年龄',
    description: '必须年满 18 岁'
  })
})

function onSubmit(data: any) {
  console.log('表单数据:', data)
}
</script>

<template>
  <MAutoForm :schema="schema" @submit="onSubmit" />
</template>
```

```vue [布局示例]
<script setup lang="ts">
const { afz } = useAutoForm()

const schema = afz.object({
  $layout: afz.$grid({ cols: 2 }),

  firstName: afz.string().meta({ label: '名' }),
  lastName: afz.string().meta({ label: '姓' }),

  email: afz.email().meta({
    label: '邮箱',
    $layout: afz.$span({ colSpan: 2 })
  })
})
</script>

<template>
  <MAutoForm :schema="schema" />
</template>
```
::

## FAQ

::accordion
:accordion-item{title="Movk Nuxt 与 Nuxt UI 有什么关系？"}
Movk Nuxt 是构建在 Nuxt UI 之上的功能增强模块。它依赖 Nuxt UI 提供的基础组件，并在此基础上提供更高级的功能，如自动表单生成、自动表格等。

你需要同时安装 Nuxt UI 和 Movk Nuxt 才能使用。
:

:accordion-item{title="为什么使用 afz 而不是原生 z?"}
`afz` 是 Zod 的增强版封装，它代理了 Zod 的方法并自动注入 AutoForm 所需的元数据。这确保了 UI 配置（如 `label`、`placeholder`）能正确传递给表单组件，并在链式调用时不会丢失。

详见 [核心概念 - useAutoForm](/docs/auto-form/core-concepts#useautoform-组合式函数)
:

:accordion-item{title="是否支持自定义控件？"}
支持。你可以通过 `type` 属性指定控件类型，或使用 `controlProps` 配置底层控件的属性。AutoForm 提供了 15+ 种内置控件类型。

未来版本将支持注册完全自定义的控件组件。
:

:accordion-item{title="支持哪些 Nuxt 版本？"}
Movk Nuxt 需要 **Nuxt 4.2.0** 或更高版本。我们专注于支持最新的 Nuxt 版本，以提供最佳的开发体验。
:

欢迎在 [GitHub Repository](https://github.com/movk/nuxt) 提交 Issue 或 Pull Request。

我们期待社区的参与和反馈！
