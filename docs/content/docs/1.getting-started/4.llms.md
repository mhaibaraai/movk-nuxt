## 什么是 llms.txt？

`llms.txt` 是一种为大型语言模型（LLM）优化的文档格式标准，它将你的文档内容转换为 AI 工具易于理解和引用的纯文本格式。这使得 AI 助手（如 Claude、ChatGPT 等）能够更准确地回答关于你项目的问题。

::callout{icon="i-lucide-sparkles" color="primary"}
Movk Nuxt Docs 默认集成了 [`nuxt-llms`](https://github.com/nuxt-content/nuxt-llms) 模块，自动为所有文档页面生成 AI 就绪文件。
::

## 如何在 AI 工具中使用

任何支持 `llms.txt` 协议的 AI 工具都可以直接访问你的文档。以下是一些使用示例：

::code-group
```text [Claude / ChatGPT]
请使用 https://nuxt.mhaibaraai.cn/llms.txt 作为参考文档，
帮我理解 @movk/nuxt 的核心概念。
```

```text [深度问答]
遵循 https://nuxt.mhaibaraai.cn/_llms-full.txt 的完整指南，
详细解释 AutoForm 的 Schema 驱动机制。
```
::

::tip
**提示**

- 使用 `/llms.txt` 进行快速问答和概念理解
- 使用 `/_llms-full.txt` 进行深度学习和详细引用
::
