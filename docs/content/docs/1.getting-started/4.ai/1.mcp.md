## 什么是 MCP？

MCP (Model Context Protocol) 是一个标准化协议，使 AI 助手能够访问外部数据源和工具。@movk/nuxt 提供 MCP 服务器，让 AI 工具能够直接访问组件信息、API 文档和使用示例。

MCP 服务器提供结构化访问 Nuxt 模块，使 AI 工具在开发过程中更容易理解和协助使用 @movk/nuxt。

## 可用资源

@movk/nuxt MCP 服务器提供以下资源供发现：

- `resource://movk-nuxt/all-components`: 浏览所有可用组件及其分类
- `resource://movk-nuxt/autoform-schema`: 浏览 AutoForm Schema 定义和控件类型

您可以在 Claude Code 等工具中使用 `@` 访问这些资源。

## 可用工具

@movk/nuxt MCP 服务器提供以下按类别组织的工具：

### 查询工具

- `list_components`: 列出所有可用的 @movk/nuxt 组件及其分类和基本信息
- `get_component`: 获取特定组件的详细文档和使用示例
- `search_composables`: 按功能或关键词搜索 Composables

## 可用提示

@movk/nuxt MCP 服务器为常见工作流提供引导式提示：

- `find_component_for_usecase`: 根据使用场景找到最适合的组件或 Composable

您可以在 Claude Code 等工具中使用 `/` 访问这些资源。

## 使用示例

配置完成后，您可以向 AI 助手提出如下问题：

- "列出所有可用的 @movk/nuxt 组件"
- "获取 MAutoForm 组件的文档"
- "DatePicker 接受哪些属性？"
- "查找表单相关的组件"
- "列出所有 Composables"
- "获取 useApiFetch 的使用示例"
- "展示所有文件上传组件"
- "我需要实现文件上传进度监控，用什么 Composable？"

AI 助手将使用 MCP 服务器获取结构化 JSON 数据，并在开发过程中为 @movk/nuxt 提供引导式协助。
