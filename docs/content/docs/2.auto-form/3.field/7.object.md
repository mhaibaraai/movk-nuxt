## 对象类型

::code-group
```ts [基础嵌套]
const schema = afz.object({
  user: afz.object({
    name: afz.string(),
    email: afz.email()
  }).meta({ label: '用户信息' })
})
```

```ts [折叠配置]
const schema = afz.object({
  profile: afz.object({
    name: afz.string(),
    bio: afz.string()
  }).meta({
    label: '个人资料',
    collapsible: { defaultOpen: true }
  })
})
```

```ts [可选对象]
const schema = afz.object({
  address: afz.object({
    street: afz.string(),
    city: afz.string()
  }).optional().meta({ label: '地址信息' })
})
```
::

## 类型化对象

::code-group
```ts [Normal Object]
interface UserInfo {
  name: string
  age: number
}

const schema = afz.object<UserInfo>()({
  name: afz.string(),
  age: afz.number()
})
// 允许额外字段，但会被移除
```

```ts [Loose Object]
const schema = afz.looseObject<UserInfo>()({
  name: afz.string(),
  age: afz.number()
})
// 允许额外字段，并保留
```

```ts [Strict Object]
const schema = afz.strictObject<UserInfo>()({
  name: afz.string(),
  age: afz.number()
})
// 不允许额外字段
```
::
