## 数字类型

::code-group
```ts [整数]
const schema = afz.object({
  count: afz.number()
    .int('请输入整数')
    .meta({ label: '数量' })
})
```

```ts [正整数]
const schema = afz.object({
  quantity: afz.number()
    .int()
    .positive('必须是正整数')
    .meta({ label: '数量' })
})
```

```ts [非负数]
const schema = afz.object({
  price: afz.number()
    .nonnegative('价格不能为负数')
    .meta({ label: '价格' })
})
```

```ts [范围限制]
const schema = afz.object({
  score: afz.number()
    .min(0, '不能小于 0')
    .max(100, '不能大于 100')
    .meta({ label: '分数' })
})
```
::
